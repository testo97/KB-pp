# KB-pp — База знаний по Проектному практикуму УрФУ

> **Что это.** Публикационный слой. Материалы для **4 аудиторий**: студенты, кураторы, партнёры, организаторы.
>
> **Источник правды.** Все материалы сюда приходят из `Pack-pp/` (приватный pack, `github.com/testo97/Pack-pp`) через протокол `pack-publish.md`.

## Структура

```
KB-pp/
├── README.md                         ← этот файл
├── КУДА-ЭТО-ПОЛОЖИТЬ.md              ← роутер: куда складывать новое
├── students/       — PlainView для студентов
├── curators/       — TechCard для кураторов, модераторов, рук. треков
├── partners/       — InteropCard для партнёров и заказчиков
└── organizators/   — AssuranceLane для организаторов
```

В каждой аудитории:
- `00-старт.md` — вход, навигация.
- `_templates/` — шаблоны карточек. **Новая запись = только копия из шаблона.**
- Папки по темам ПП: `01-запуск-семестра/`, `02-контрольные-точки/`, `03-защиты/`, `04-оценка/`, `05-коммуникация/`, `06-спорные-ситуации/`, `роли/`.

## Правила

1. **Содержимое приходит только через `pack-publish`.** Редакции вручную — только по согласованию.
2. **Новая запись — копия из `_templates/`.** Без шаблона не писать.
3. **Без FPF/SPF-жаргона** в пользовательском тексте (кроме organizators, где допустимо).
4. **Темы — сверху, тип карточки — внутри.** Структура подчинена реальным задачам аудитории.
5. **Версионирование.** Каждая карточка имеет дату `updated` в шапке.
6. **Никаких drafts здесь.** Черновики — в `Pack-pp/` (канон). Сюда приходит готовое.
