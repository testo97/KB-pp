# Куда это положить

Короткий роутер, если непонятно, в какую папку что идёт.

## По аудитории

| Для кого это | Папка |
|--------------|-------|
| Студент — чтобы понять, что делать, куда идти, что от него ждут | `students/` |
| Куратор / модератор / рук. трека — полная механика и инструкции | `curators/` |
| Партнёр / заказчик — точки взаимодействия, что нужно от них и для них | `partners/` |
| Организатор — чек-листы, регламенты, метрики для контроля | `organizators/` |

Одно и то же знание обычно публикуется на **несколько** поверхностей — в своей проекции.

## По теме (внутри каждой аудитории)

| Тема | Папка |
|------|-------|
| Как стартует семестр, как формируются команды, выбор проекта | `01-запуск-семестра/` |
| Что происходит в середине семестра, как сверяемся | `02-контрольные-точки/` |
| Защиты — формат, критерии, подготовка | `03-защиты/` |
| Оценка — кто, как, когда ставит | `04-оценка/` |
| Переписка, встречи, каналы связи | `05-коммуникация/` |
| Что делать, если что-то пошло не так | `06-спорные-ситуации/` |
| Описания ролей участников ПП | `роли/` |

## По типу карточки

| Если пишете… | Шаблон |
|--------------|--------|
| Пошаговую инструкцию («как сделать X») | `_templates/инструкция.md` |
| Описание роли (что от меня ждут) | `_templates/роль.md` |
| Правило («всегда», «нельзя») | `_templates/регламент.md` |
| Термин / определение | Общий глоссарий (будет добавлен) |

Новый файл — **обязательно копия из `_templates/`**. Не переписывайте структуру «от руки».
